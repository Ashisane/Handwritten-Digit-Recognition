How It Works

01

Forward Pass

Each pixel feeds into 128 hidden neurons via a weighted sum.

def learn(data, weights):
    return np.dot(data, weights)

def hidden_unit(data, weights):
    return activation_function(learn(data, weights))
02

ReLU Activation

Negative signals are zeroed out. Only useful features survive.

def activation_function(feature):
    return(max(0, feature))
03

Softmax → Prediction

10 output neurons squeezed into probabilities. The highest wins.

def softmax(features):
    updated_features = []
    for feature in features:
        updated_features.append(np.e ** feature)
    total_sum = np.sum(updated_features)
    return [i / total_sum for i in updated_features]
04

Backpropagation

Error flows backward, adjusting every weight to be slightly less wrong.

def error_l2(p, l):
    e = []
    for i in range(len(p)):
        if i == l: e.append(1-p[i])
        else: e.append(-p[i])
    return np.array(e)

def update_weights(cost, weights, lr=0.01):
    return weights + cost * lr
